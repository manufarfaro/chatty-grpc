Some dummy chat app to test around grpc technology in deno.

## Quick Start

```bash
# Run the app
deno run src/main.ts

# Run in watch mode
deno task dev
```

## Testing

```bash
# Run tests
deno test

# Run tests with coverage
deno test --coverage
```

## Build

```bash
# Compile for your machines only
deno task compile

# Compile for all platforms
deno task compile:all
```
